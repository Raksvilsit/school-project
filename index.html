Практическая демонстрация работы интернета

История и технологии глобальных сетей

1. Пакетная передача данных

Сообщение разбивается на отдельные пакеты, которые передаются независимо друг от друга и собираются на стороне получателя.

2. Маршрутизация пакетов

Каждый пакет может проходить через разные сетевые узлы (маршрутизаторы), что повышает устойчивость сети.

3. DNS — преобразование имени в IP

DNS переводит доменные имена, удобные для человека, в числовые IP-адреса, которые используются компьютерами.

4. Клиент — сервер

Браузер (клиент) отправляет запрос серверу, а сервер возвращает ответ в виде веб-страницы.
